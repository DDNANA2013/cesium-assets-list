Using this tileset in Cesium

// Get your own Bing Maps API key at https://www.bingmapsportal.com and include the line
// below in your application, prior to creating a Cesium scene:
Cesium.BingMapsApi.defaultKey = 'put your API key here';
	
// Applications that use CesiumWidget or Viewer automatically use Bing Maps for imagery by default.
// In other applications:
var scene = ...;
scene.imageryLayers.removeAll(); // optional
scene.imageryLayers.add(new Cesium.BingMapsImageryProvider({
	url : '//dev.virtualearth.net'
}));
